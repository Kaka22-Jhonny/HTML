Perguntas frequentes Sobre html.

Respostas sempre abaixo das perguntas!

  • Como fazer listas?

    • As tags usadas para fazer uma lista ordenada são "ul" e "li".
    A tag "ul" é usada para definir a lista e a tag "li" é usada para cada item da lista.
